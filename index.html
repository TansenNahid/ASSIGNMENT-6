  • What are the difference between Var, Let, Const?

    Var

    The var keyword is the first one used in JavaScript to declare a variable. When a variable is decclared with var, we can assign the reassign the value using var keyword. Ex: var a =5, var a = 8

    Let

    The let keyword is an improved version of the var keyword. When we declare a variable first time we have to use Let keyword. Then if any necessary to reassign the same variable, we need to write the varible name and the value we are ressigning. Ex: let a =5, a = 8

    Const

    The only difference between the let and const keywords is that the const keyword cannot be updated by the user. when we assign a variable using const keyword we can not change or update the value of the variable. if we try it will give an error. Ex: const a =5

The arrow functions can never be used as constructor functions because they are only callable, not constructible. As a result, the new keyword cannot ever be used to invoke them. Arrow functions lack their own this, in contrast to normal functions. Throughout the function's lifetime, the value of this inside an arrow function stays constant and is always bound to the value of this inside the nearest non-arrow parent function.

  • Why we should use template string?

    A powerful component of contemporary JavaScript introduced in ES6 are template strings. Instead than requiring to concatenate like in earlier JavaScript versions, it enables us to insert/interpolate variables and expressions into strings.We insert a variable or expression by adding a dollar sign $ and curly braces {} into the string `${variable}`. This is much better than the alternative in old JavaScript where we had to concatenate a string.
